# QDAuth

Quick and Dirty Authentication (QDAuth) is a federated, session-less,
token-based authentication framework. It authenticates users with
either a local identity provider or an external identify provider (social login
with Google, Microsoft, GitHub, etc), and issue access tokens to access
resources.

## Features

- Local Login
- Social Login with external Identity providers like Google and Microsoft
- Spring Authorization Server
- JWT Access Tokens
- Refresh token management
- Accounts resource

## Technology

- Docker
- Spring Boot
- Spring MVC
- Spring Security
- Spring Data JPA and Hibernate
- JOSE
- OAuth2
- OIDC
